# dBsource 官网 — 管理员使用说明书

> **适用：** 市场、运营、销售助理  
> **后台：** http://127.0.0.1:3003/admin  
> **版本：** v2.1 · 2026年6月

---

## 一、登录

1. 打开 http://127.0.0.1:3003/admin/login  
2. 输入管理员口令（`ADMIN_TOKEN`，由技术配置）  
3. 进入总览页，点击分区卡片编辑  

**配图：** 登录页截图（Logo + 口令框）

---

## 二、通用操作

| 操作 | 说明 |
|------|------|
| 搜索 | 顶部按名称、型号过滤 |
| 未保存 | 修改后显示黄色「未保存」标签 |
| 保存并发布 | 底部金色按钮，同步官网 |
| 预览页面 | 右上角跳转官网对应页 |
| 刷新 | 重新拉取服务器数据 |

**必做：** 改完务必点「保存并发布」。

---

## 三、产品中心

| 字段 | 说明 |
|------|------|
| 型号 | 唯一标识，如 V212 |
| 名称/简介/规格 | 中英文 |
| 封面图 | 单张，右上角 × 删除 |
| 产品图集 | 网格缩略图，每张右上角 × 删除；「追加图片」上传 |

流程：搜索 → 展开 → 编辑 → 保存 → 预览 `/products`

**配图：** 后台图集网格 + 前台产品详情页

---

## 四、下载中心

| 字段 | 说明 |
|------|------|
| Tab / 子分类 | 软件或画册、导航筛选 |
| 下载文件 | ZIP/PDF，带进度条，最大 512MB |
| 文件大小 | 上传后自动填写（只读） |
| 下载文件名 | 自动填写，用户下载保留原名（只读） |
| 排序 ID | 分享链接中的数字 |

分享链接：`/api/downloads/{排序ID}/file`，打开即下载。

**配图：** 下载列表 + 上传进度条

---

## 七、图片规则

- **单图：** 预览右上角 × 删除  
- **图集：** 每张独立删除，不会误删其他图  
- **手机预览：** 访问 `http://<电脑IP>:3003`，图片自动代理加载  

---

## 八、新增 / 删除

产品、下载、案例、系列、二维码支持「+ 新增」与「删除」（需确认）。

---

## 九、常见问题

| 问题 | 处理 |
|------|------|
| 保存失败 | 确认 Strapi 已启动，点刷新重试 |
| 图片不显示 | 保存后等几秒刷新官网 |
| 下载大小不对 | 重新上传，自动校正 |
