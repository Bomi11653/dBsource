# dBsource 品牌官网 — 飞书项目介绍

> **品牌：** 东莞新声电子科技有限公司 · dBsource  
> **版本：** v2.1 · 2026年6月  
> **代码仓库：** [github.com/Bomi11653/dBsource](https://github.com/Bomi11653/dBsource)

---

## 一、项目是什么？

**dBsource 官网**是专业音响品牌的数字化门户：面向客户展示产品、案例与资料；面向销售承接询盘与智能选型；面向运营提供零代码内容管理。

| 维度 | 说明 |
|------|------|
| 前台 | 中英文双语，电脑 + 手机自适应 |
| 内容后台 | http://127.0.0.1:3003/admin 日常改文案、图片、产品 |
| 数据后台 | Strapi CMS（1337）统一存储与发布 |
| 智能能力 | AI 顾问、三场景选型器、全局搜索 |

## 三、核心页面

| 页面 | 路径 | 说明 |

## 五、建设亮点

1. WebGL 首页声波、页面过渡动画、案例滚动叙事  
2. 系列 mega 导航、产品详情、三场景选型器  
3. 下载中心：大文件上传、分享直链、保留原始文件名  
4. DeepSeek AI 顾问，页面感知，销售导流  
5. 自建中文后台：搜索、图集删改、预览、直传加速  
6. Strapi 异常时自动降级，站点不白屏  

---

## 六、技术栈

Next.js 14 · TypeScript · Tailwind · Strapi 5 · Three.js · DeepSeek

```bash
git clone https://github.com/Bomi11653/dBsource.git
cd dBsource && npm install && npm run dev:mobile
```
