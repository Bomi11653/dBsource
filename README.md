# dBsource Pro

东莞新声电子科技有限公司 — 专业音响品牌官网（Next.js 14）。

## 本地预览

```bash
npm install
npm run build
npm run start
```

浏览器打开 http://127.0.0.1:3003

Windows 可双击上级目录的 `修复预览崩溃.bat`。

## 技术栈

- Next.js 14 · TypeScript · Tailwind CSS
- Three.js / React Three Fiber（首页声波）
- Framer Motion（案例滚动叙事）
- Mock CMS，可接 Strapi

## 环境变量

复制 `.env.example` 为 `.env.local`（可选）。
