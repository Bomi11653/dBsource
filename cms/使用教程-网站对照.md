## 一、先搞懂三件事

| 概念 | 中文 | 说明 |
|------|------|------|
| **Content Type** | 内容类型 | 一种内容的「表格」，如产品、案例 |
| **Entry** | 条目 | 表格里的一行，如「AK Live House」案例 |
| **Publish** | 发布 | 只有点 **Publish** 后，官网才能读到（将来接 API 时） |

**双语怎么存？**  
每个字段分两个框：`xxxZh` = 中文，`xxxEn` = 英文。官网切换语言时会自动显示对应字段。

**后台界面变中文：**  
登录后 → 左下角头像 → **Profile** → **Experience** → **Interface language** → 选 **中文（简体）**。

## 二、Strapi 菜单 vs 官网页面对照

```
Strapi 后台                          官网页面
─────────────────────────────────────────────────────────
① 产品 Product          →    /products  产品中心
② 案例 Case             →    /cases     工程案例
③ 下载 Download         →    /downloads 下载中心
④ 场景 Scene            →    首页「应用场景」3 张图
⑤ 二维码 QRCode         →    页脚 + 联系我们页
⑥ 关于 AboutSection     →    /about     关于我们配图
```

### 详细位置对照表

| Strapi 类型 | 官网位置 | 客户常改什么 |
|-------------|----------|--------------|
| **Product** | 产品中心列表、产品详情页 | 型号名、简介、参数、产品图 |
| **Case** | 工程案例列表、案例详情、首页精选 | 案例标题、描述、封面、图集 |

## 四、日常操作步骤（手把手）

### 新建或修改一条内容

1. 左侧 **Content Manager**（内容管理器）
2. 点击类型，如 **② 案例 Case**
3. 点右上角 **Create new entry**（创建新条目）或点已有条目编辑
4. 填写 **Zh** 中文字段和 **En** 英文字段
5. 图片字段点 **Upload** → 从电脑选择或从 **Media Library** 选已有图
6. 右上角切换 **Draft** → 点 **Publish**（发布）
7. 保存后官网需接 API 才会自动更新（目前官网仍读本地数据）

### 上传图片到媒体库

1. 左侧 **Media Library**（媒体库）
2. **Add new assets** → 选择图片
3. 在编辑条目时，点图片框 → **Media Library** → 选中图片

### 常见按钮

| 英文 | 中文 |
|------|------|
| Save | 保存草稿 |
| Publish | 发布（对外可见） |
| Unpublish | 取消发布 |
| Delete | 删除 |

1. **QRCode**（3 条，最简单）→ 练手
2. **Scene**（3 条）→ 首页场景
3. **AboutSection**（9 条）→ 关于页配图
4. **Case**（6 条）→ 案例文案+图

6. **Product**（55 条）→ 工作量大，可分批

## 七、启动后台

双击 `启动后台.bat` 或：

```powershell
cd C:\Users\Administrator\Desktop\Strapi
npm run develop
```

有问题先看终端报错，或联系技术人员协助接官网 API。
